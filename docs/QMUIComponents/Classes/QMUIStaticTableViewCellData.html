QMUIStaticTableViewCellData

@interface QMUIStaticTableViewCellData : NSObject

一个 cellData 对象用于存储 static tableView（例如设置界面那种列表） 列表里的一行 cell 的基本信息，包括这个 cell 的 class、text、detailText、accessoryView 等。
